BIENVENIDO A ESTA PÁGINA WEB

¿De qué trata esta pagina web?

Este es un prototipo de una pagina web donde se va a aprender a crear paginas web. Entre ellos están:

  1. Diseño de titulo.
  2. Implementación de imagenes.
  3. Implementar paginas interactivas.
  4. Implementación de interactividad (Scripts).

Diseño de un titulo

¿Qué es lo que se necesita saber?

Para poder implementar un titulo y su diseño, es facil:
Primero se tiene que iniciar con "h1" haciendo referencia a head 1 o cabeza 1 y terminar con "/h1"
Para agregar diseños o ciertas especificaciones, por ejemplo, colocar el titulo o texto centrado a la página, se puede implementar:

style="text-aling: center"

Implementación de imagenes

¿Qué es lo que se necesita saber?

Para poder implementar una imagen y diseño:
Primero hay que agregar la imagen al repositorio o al programa, en este caso, yo tengo un repositorio llamado "proyecto pagina web" y
en ese repositorio tengo una carpeta llamada "imagenes", en siguiente, se tiene que implementar la codificación:

div class="bloque2"
....h1 style="text-align: center;" ESTA PÁGINA FUE CREADA POR: /h1
....img src="imagenes/imagen1.png" alt="Descripción de la imagen"
/div

aplicando un diseño llamado bloque2 donde se centra la imagen y se hace pequeña aplicando un bloque con color de fondo naranjo:

style
....img {
........display: block;
........margin: 0 auto;
........width: 150px;
........height: 200px;
....}
/style
